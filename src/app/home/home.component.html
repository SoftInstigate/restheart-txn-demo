1- Start RESTHeart (v >= 5.1)

using Docker

Alternatively get the binary at https://restheart.org/get

$ mkdir restheart && cd restheart
$ curl https://raw.githubusercontent.com/SoftInstigate/restheart/master/docker-compose.yml --output docker-compose.yml
$ docker-compose up -d

2 - Create db and collections

using curl

$ curl -u admin:secret -v -X PUT :8080/
$ curl -u admin:secret -v -X PUT :8080/palettes
$ curl -u admin:secret -v -X PUT :8080/colors
